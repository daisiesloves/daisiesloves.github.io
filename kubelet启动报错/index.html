# kubelet 时启动报错

检查kubelet配置文件kubelet-config.yml，发现某行少一个空格，导致服务启动异常

# 错误日志如下：

journalctl -u kubelet --no-pager
journalctl 的 -u 参数可以指定服务进行过滤，这样可以屏蔽掉其他无关日志。--no-pager 参数可以一次性输出日志，当然如果你只是在线查看，则可以不用这个参数，只是输出日志受到屏幕宽度限制，需要通过方向键滚动
