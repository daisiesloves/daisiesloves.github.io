# 部署 etcd 集群，启动时报错 "publish-timeout":"7s","error":"etcdserver: request timed out

部署好etcd集群，我是先启动一个节点测试，结果启动失败报错。原因是etcd.service中指定了其它主机节点IP，单独启动一个节点，etcd会启动失败，把其它节点etcd.service都restart下即可。
