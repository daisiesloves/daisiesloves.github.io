# Mysql 数据库表没有创建分区导致 Spark 任务异常

Spark向Mysql插入数据，因Mysql表没有创建分区，导致数据写入失败。期间通过Web页面查看spark任务，并未出现相关异常。原因是java程序中定义了try catch机制，当try catch捕获到异常时，程序不会异常退出。

# Spark 日志报错如下：

spark mysql try catch
