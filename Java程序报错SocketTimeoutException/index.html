# Java 应用连接数据库查询时 java.net.SocketTimeoutException 报错

应用程序连接数据库时，连接超时查询数据失败。出现这个问题的原因是OKhttp没有留足够的时间连接和处理数据，造成了超时的情况。

# 解决方案：

1. 将超时时间延长即可
2. 连接数据库配置多个节点
